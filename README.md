# ReportsGenerator

Generation of reports from a food delivery application. Deliveries are stored in a csv file.

- [x] - Which user placed the most orders
- [x] - Which food was most requested
- [x] - Complete report of all users and their order quantities
- [x] - Complete report of all foods and order quantities
